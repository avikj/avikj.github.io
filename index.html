// Projects

SwiftAssist

Built at Los Altos Hacks

SwiftAssist is an application that revolutionizes health emergency response using crowdsourcing. For example, if someone is having an allergic reaction and they do not have their epinephrine injector (EpiPen), they can press a button on their Pebble smartwatch or iPhone to notify nearby users who can help them. These users recieve an alert on their phones and can view directions to the location of the emergency, along with what type of emergency it is. I built the Pebble smartwatch app using the Pebble.js SDK.

Best use of Microsoft Technologies Winner

Flappy Nerd

Built at CU Hacks

Flappy Nerd is an immersive Flappy Bird based game that students can use to study. It uses data scraped from Quizlet, to allow users to choose sets that they want to practice with. Unlike Flappy Bird, the pipes in Flappy Nerd have two gaps. The user must maneuver the bird through the gap with the correct answer to a question that is displayed below, to continue playing. I worked on the actual game functionality and graphics. I also played a role in parsing the data retrieved from Quizlet.

2nd Place for Best Educational Hack

QuickChat

Built in my room.

QuickChat is a quick online chatting application that allows users to send uniquely generated URL's to their friends and broadcast messages to anyone who's looking at the URL. It was built with the Meteor platform, using Materialize.css for the user interface.
