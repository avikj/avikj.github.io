Flappy Nerd

A better way to study.

Flappy Nerd is a desktop FlappyBird clone where users can search Quizlet sets by ID or by keywords, and can then choose a set to start the game. The gameplay is similar to that of Flappy Bird. Under each pipe column, there is a study term. Unlike traditional Flappy Bird, each pipe column has two gaps, one with the correct definition, and one with a randomly chosen incorrect definition. The user must maneuver the bird through the correct gap to earn points. Using Flappy Nerd is an immersive alternative to studying off of flashcards, and has proven to be more fun and effective, especially for vocabulary sets. This began as a school project, but was rebuilt completely at CU Hacks, with the added functionality of using Quizlet sets. It won second place in the educational hack category at CU Hacks.

Vroom

Drive your education.

Vroom is a web application that is meant to simulate a classroom environment for students and teachers. Teachers can create an account and then create classrooms for each of their subjects. Students can see these classrooms in the class list, and join based on what they want to review. The classroom takes place in a moderated chat environment, where the teacher can choose which messages are displayed to the whole class, and when. This project was targeted at summer classes. Vroom was built at HackingEDU View the Github Repo here
